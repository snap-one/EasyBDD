# Media Tests

Tests for media streaming and control systems.

## Devices Covered

- **Media Players** - Streaming devices and players
- **Video Matrix** - Video distribution systems
- **AV Receivers** - Audio/video receivers
- **Set-Top Boxes** - Cable/satellite receivers
- **Smart TVs** - Network-connected televisions

## Test Categories

### Playback Control
- Play/pause/stop
- Volume control
- Input selection
- Channel changing

### Streaming
- Stream quality
- Buffering behavior
- Format support
- Network streaming

### Configuration
- Resolution settings
- Audio output format
- Network configuration
- Source management

### Integration
- Control system integration
- API control
- IR/RS232 control
- HDMI-CEC
