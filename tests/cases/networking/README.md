# Networking Tests

Tests for network infrastructure devices.

## Devices Covered

- **Switches** - Managed and unmanaged network switches
- **Routers** - Network routers and gateways
- **Access Points** - Wireless access points
- **Controllers** - Network management controllers

## Test Categories

### Configuration
- VLAN configuration
- Port settings
- QoS policies
- Access control lists

### Connectivity
- Port link status
- DHCP functionality
- DNS resolution
- Gateway routing

### Performance
- Throughput testing
- Latency measurements
- Packet loss analysis
- PoE power delivery

### Monitoring
- Device status
- Port statistics
- Traffic analysis
- Error logging

### Firmware
- Firmware updates
- Configuration backup/restore
- Factory reset
- Boot sequence
