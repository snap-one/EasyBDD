# Power Management Tests

Tests for power management devices and systems.

## Devices Covered

- **WattBox** - IP-controlled power distribution units
- **PDUs** - Power distribution units
- **UPS** - Uninterruptible power supplies

## Test Categories

### Power Control
- Outlet on/off control
- Power cycling
- Sequential power-up
- Group power management

### Monitoring
- Power consumption tracking
- Voltage monitoring
- Current draw measurements
- Temperature monitoring

### Configuration
- Outlet naming and grouping
- Auto-reboot settings
- Power-on delay configuration
- Network settings

### Integration
- OVRC integration
- API authentication
- Event notifications
- Firmware updates
