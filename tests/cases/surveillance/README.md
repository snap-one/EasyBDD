# Surveillance Tests

Tests for security cameras and recording systems.

## Devices Covered

- **IP Cameras** - Network-connected security cameras
- **NVRs** - Network video recorders
- **DVRs** - Digital video recorders
- **Video Analytics** - AI-powered video analysis

## Test Categories

### Camera Control
- PTZ (pan/tilt/zoom) control
- Focus adjustment
- Preset positions
- Tour sequences

### Video Quality
- Resolution settings
- Frame rate
- Compression quality
- Night vision

### Recording
- Continuous recording
- Motion-triggered recording
- Event-based recording
- Storage management

### Motion Detection
- Detection zones
- Sensitivity settings
- Alert notifications
- False positive handling

### Integration
- OVRC integration
- Mobile app access
- Alert delivery
- Third-party integration
